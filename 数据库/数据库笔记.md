# CHAR 和 VARCHAR 的区别

**CHAR：**定长字符串

* 长度范围：0 - 255
* CHAR(20)，如果存储一个长度不足20的字符，MySQL会用空格填充后面剩余的字符。



**VARCHAR：**变长字符串

* 长度范围：0 - 65535 
* VARCHAR(20)，如果存储一个长度不足20的字符，系统根据实际存储的数据量来分配合适的存储空间。

# 主键约束和唯一约束的区别

**主键约束（PRIMARY KEY)：**

- PRIMAEY KEY 主键，是唯一的，一张表只能有一个主键
- AUTO_INCREMENT 一定要和主键连用
- 但主键不一定要和 AUTO_INCREMENT 连用
- 主键一定是非空的 NOT NULL

**唯一约束（UNIQUE KEY）：**

- UNIQUE KEY 唯一约束
- 唯一约束可以保证数据的唯一性
- 唯一约束的字段可以为空
- 每张数据表可以有存在多个唯一约束
